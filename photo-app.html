Photoshare Web App

Back

Course: CS142: Web Applications

Time: 4 weeks

Skills: JavaScript, CSS, HTML, ReactJS, NodeJS, ExpressJS

About: In this project, I built a full-stack photo sharing web application. I built the front-end with React following the Model-View-Controller (MVC) design pattern. I also integrated AJAX to fetch data from the server and established the database using MongoDB. Features include:

  • Liking/Unliking photos and sorting by most liked
  • Deleting comments, photos, and your account
  • Displaying a user activity log to show all users' most recent actions
  • Displaying a user's most recent photo and photo with the most comments
